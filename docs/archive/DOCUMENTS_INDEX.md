## 🚀 DÉMARRAGE RAPIDE

```bash
# 1. Démarrer le projet
python manage.py runserver

# 2. Faire un backup
python manage.py backup_data

# 3. Cloner sur autre PC (voir REPONSE_CLONE_FACILE.md)
git clone [repo]
python manage.py restore_data backup.tar.gz --force
```
